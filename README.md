BIG DATA PROJECT: АНАЛИЗ ПОКУПАТЕЛЬСКОГО ПОВЕДЕНИЯ

проект посвящен анализу больших данных о продажах. Основная цель - обработка сырых данных, вычисление бизнес-метрик и визуализация результатов с помощью Python

ОПИСАНИЕ
В рамках проекта выполняется
1. Загрузка и очистка данных.
2. Расчет метрик (выручка, средний чек, стоимость позиции).
3. Визуализация распределений.
4. Работа с базой данных через SQLAlchemy.

СТРУКТУРА ПРОЕКТА
bigdata_project/
    dashboards/
        analysis.ipynb    (Основной ноутбук с анализом и графиками)
    data/                 (Папка с исходными данными)
    requirements.txt      (Список библиотек для установки)
    README.md             (Документация проекта)

УСТАНОВКА И ЗАПУСК

1. Переход в папку проекта:
cd C:\Users\user\Desktop\bigdata_project

2. Создание виртуального окружения:
python -m venv bigdata_venv

3. Активация окружения (Windows):
bigdata_venv\Scripts\activate

4. Установка зависимостей:
pip install -r requirements.txt
