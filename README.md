Анализ покупательского поведения (Big Data Project)

Проект представляет собой распределенную систему обработки данных (ETL), которая анализирует транзакции интернет-магазина, выполняет RFM-сегментацию клиентов и визуализирует результаты на интерактивном дашборде.

Технологический стек

* Оркестрация: Prefect 2.x для управления задачами и рабочими процессами 

* Обработка данных: Dask для параллельных вычислений и агрегации

* Хранение: PostgreSQL как реляционная база данных 

* Визуализация: Streamlit и Plotly для построения интерактивного дашборда

* Инфраструктура: Docker и Docker Compose для развертывания системы


Структура проекта

* data/: Исходный датасет (online_retail.csv) 

* flows/: Скрипты Prefect (etl_flow.py)

* dask_jobs/: Обработка данных на Dask и функции взаимодействия с БД

* dashboards/: Визуализация на Streamlit

* docker-compose.yml: Конфигурация запуска инфраструктуры

* requirements.txt: Список зависимостей Python 

* report.docx: Отчет по проекту


Запуск проекта

1. Подготовка
Убедитесь, что установлены Docker и Docker Compose
2. Переход в папку проекта:
cd C:\Users\user\Desktop\bigdata_project
3. Создание виртуального окружения:
python -m venv bigdata_venv
4. Активация окружения (Windows):
bigdata_venv\Scripts\activate
5. Развертывание инфраструктуры
Запустите базу данных и сервис дашборда командой:
docker-compose up -d
5. Запуск ETL-процесса
Установите зависимости и запустите пайплайн обработки данных:
pip install -r requirements.txt
python flows/etl_flow.py

Пайплайн включает три основные задачи: Extract (чтение данных), Transform (RFM-анализ в Dask) и Load (сохранение в Postgres). 

6. Просмотр результатов
После завершения работы скрипта откройте дашборд в браузере по адресу http://localhost:8501

Описание обработки (RFM-анализ)
В ходе обработки с помощью Dask для каждого уникального клиента рассчитываются следующие метрики:

1. Recency (Давность): количество дней с момента последней покупки 


2. Frequency (Частота): общее количество транзакций клиента


3. Monetary (Деньги): суммарная выручка от всех покупок клиента
